# Kanban Capstone

## Project Overview

An advanced Kanban task organizer. Capstone Project for PDX Code Guild Class Raven.

## Functionality

Initial Features:

- Add a kanban
- Add a task
- Assign task to a column
- Add a comment

Extra Features:

- Add subtasks
- Add subcomments
- Add tags
